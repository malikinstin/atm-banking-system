ATM Banking System

Overview

This project is a console-based ATM Banking System developed in C++. It is currently a work in progress and serves as a practical exercise in applying object-oriented programming and software development concepts.

The current version allows users to check account balances through a simple menu-driven interface. Additional banking features will be implemented in future updates.

Current Features

- Balance checking
- Menu-driven user interface
- User input handling

Technologies Used

- C++
- Standard Library

How to Run

1. Clone the repository:
   
   git clone <repository-url>

2. Navigate to the project directory:
   
   cd atm-project

3. Compile the program:
   
   g++ main.cpp -o atm

4. Run the program:
   
   ./atm

Planned Features

- User authentication (PIN verification)
- Cash deposits
- Cash withdrawals
- Fund transfers
- Transaction history
